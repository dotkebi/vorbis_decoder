# Licensed fixture sources

`salamander_phrase.flac` is a three-second arpeggio assembled from the medium
velocity C4, D#4, F#4, and A4 recordings in Salamander Grand Piano v3.

- Original author and recording: Alexander Holm
- License: Creative Commons Attribution 3.0 Unported (CC BY 3.0)
- Original archive: <https://archive.org/details/SalamanderGrandPianoV3>
- Instrument repository: <https://github.com/sfzinstruments/SalamanderGrandPiano>

The cropped/concatenated source remains available under CC BY 3.0. It is
committed so fixture regeneration has no dependency on sibling projects or an
external recording download.
